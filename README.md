Stellar Expedition: Journey Beyond Earth

Project Description
A responsive and visually engaging landing page for a space exploration themed website. The project focuses on modern UI design and smooth user experience with an immersive space atmosphere.

Technologies Used
HTML5 for structure
CSS3 for styling and layout
Google Fonts including Orbitron, Inter, and Roboto
Video background for immersive visual experience

Features
Interactive and modern UI components
Full screen video background for space theme
Responsive design for different screen sizes
Clean typography using Google Fonts
Visually appealing landing page layout

Design Focus
The project emphasizes futuristic space aesthetics, smooth visual hierarchy, and user engagement through animation and multimedia integration.

Author
Frontend landing page project created for practice in modern web design and UI development.
